## 三、优先级建议

| 优先级 | 功能 | 原因 |
|--------|------|------|
| 🔴 P0 | 多日期区间选择 | 用户痛点：只能选一个终点，无法规划多段 |
| 🔴 P0 | 目标导向计算 | 用户痛点：需要反向计算"要攒多久" |
| 🟡 P1 | 自定义规则编辑器 | 扩展性：活动变化频繁，预设跟不上 |
| 🟡 P1 | 深色/浅色主题 | 视觉：部分用户偏好浅色 |
| 🟡 P1 | 数据导出 | 分享：用户想保存/分享规划结果 |
| 🟢 P2 | 多账号管理 | 便利：大小号分开计算 |
| 🟢 P2 | PWA 离线支持 | 可用性：弱网环境也能用 |
| 🟢 P2 | 移动端手势 | 体验：手机端滑动更自然 |
| ⚪ P3 | 云端同步 | 进阶：跨设备使用 |
| ⚪ P3 | 单元测试 | 质量：保证核心逻辑正确 |
| ⚪ P3 | 国际化 | 受众：覆盖海外玩家 |
